Udacity Nano degree IPND

Rajeev Midha | 2020/07/02

This nano degree gave me a solid foundation in programming. It is a challenging course for newbies and is very practical The first project requires you to create a simple web page using basic CSS principes, the second project requires you to create a simple game in python and the third was to create a simple web app using plain vanilla javacript. The projects provided me with the right level of challenge. I think all people who want to learn to code but do not know where to start to should with this course.
